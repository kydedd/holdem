Description
-------------------------------------------------------------------------------
Thank you for downloading Texas Hold'em 0.1.0!  This Texas Hold'em 
program is a text based game that simulates a Sit and Go No Limit Texas 
Hold'em poker tournament with you and eight other non-real players using the 
command line.  TexasHoldem.jar is the main Texas Holdem application.  You can 
double-click this file to run the game application.  You can also run the 
application on the command line.  Navigate to the TexasHoldem directory and 
enter the command:

  java -jar TexasHoldem.jar


Help
-------------------------------------------------------------------------------
Information on the rules of No Limit Texas Hold'em can be found on the web at:

  http://www.pokerstars.com/poker/games/texas-holdem/


Editing Source Code
-------------------------------------------------------------------------------
The Java source code is in the directory named src.  You can edit this source 
code in an integrated development environment such as Eclipse or Netbeans by 
adding a copy of this directory to your project.  A Windows utility script 
(build.bat) is included in this directory.  When executed on the command line 
this script will create an executable jar file TexasHoldem.jar.
